# Method of Loci - 업데이트 내역 (2026-06-19)

오늘 진행된 'Method of Loci (기억의 궁전)' 프로젝트의 주요 업데이트 내용입니다. 3D 시각화와 물리적인 상호작용(UX)을 대폭 개선하여 실제 서재와 같은 경험을 제공하는 데 집중했습니다.

## 1. 3D 책 회전 엔진 및 표지 모드
- **3D 입체화 (`preserve-3d`)**: 단순한 2D 사각형이었던 책 컴포넌트를 입체 상자 형태로 재구성하여 책등(Spine)과 표지(Cover) 면을 동시에 렌더링하도록 구현했습니다.
- **마우스 호버(Hover) 효과**: 책등 뷰에서 특정 책에 마우스를 올리면, 해당 책이 앞으로 당겨지며 부드럽게 75도 회전하여 표지를 살짝 보여주는 물리 애니메이션을 적용했습니다.
- **뷰 모드 토글 (책등 / 표지 모드)**: 화면 우측 상단의 뷰 모드 아이콘(바둑판 / 책꽂이 모양)을 클릭해 전체 책을 일제히 정면 표지 모드로 전환하거나 원래대로 돌려놓을 수 있습니다.

## 2. 외부 링크 썸네일 표지 자동 적용
- **Microlink API 연동**: 책 내용에 웹사이트 링크를 첨부할 경우, 카카오톡/디스코드처럼 링크의 메타데이터(Open Graph 이미지, 제목, 설명)를 카드 형태로 불러오는 기능을 구현했습니다.
- **표지 썸네일 대체**: 책에 외부 링크가 존재할 경우, 사용자가 올린 일반 이미지보다 외부 링크의 메인 썸네일을 최우선으로 책 표지 전면에 렌더링하도록 로직을 수정했습니다.

## 3. UI/UX 및 애니메이션 최적화
- **책 밀어내기 (Hover Expansion)**: 책에 마우스를 올려 3D로 회전할 때 주변 책들이 자연스럽게 밀려나도록 공간(width)을 동적으로 확보했습니다. `flex-shrink: 0` 속성을 추가해 책이 많아져도 찌그러지지 않고 확실하게 옆으로 밀어냅니다.
- **Z-Index 및 클릭 버그 수정**: 표지 모드에서 확대된 책이 이웃한 책에 가려지는 문제를 Z-Index 동적 할당으로 해결했으며, 3D 회전 상태에서 클릭 이벤트가 무시되는 브라우저 고질적 버그를 최상위 Wrapper에 onClick을 위임하여 완벽히 해결했습니다.
- **세로 쓰기(`vertical-rl`) 원복**: 책등의 제목을 한글은 똑바로 서서 세로로 쌓이고, 영문은 누워서 세로로 읽히도록 동양권 서재의 자연스러운 세로 쓰기 레이아웃으로 되돌렸습니다.

## 4. 책장(Shelf) 커스텀 및 드래그 앤 드롭
- **책장 순서 변경 (Drag & Drop)**: 상/하 화살표 대신 책장 좌측의 손잡이(Grip) 아이콘을 잡고 드래그하여 책장의 위치를 통째로 바꿀 수 있게 되었습니다. (Framer Motion `Reorder` 사용)
- **드래그 중 텍스트 선택 방지**: 책장 손잡이를 드래그할 때 주변 글씨(책장 이름, 권수)가 파란색으로 블록 지정되는 불편함을 `user-select: none`으로 차단했습니다.
- **책 색상 변경 기능**: 책 수정 화면(BookPanel)에 8가지 프리셋 컬러와 무한 커스텀 컬러 피커를 추가하여 개별 책의 테마 색상을 자유롭게 변경하고 저장할 수 있게 되었습니다.

## 5. 책장 가로 스크롤 (Carousel) 구현
- **화살표 내비게이션**: 한 책장에 책이 가로폭을 초과하여 추가될 경우, 책장을 뚫고 나가지 않고 스크롤되도록 구조를 개편했습니다. 책장 양 끝에 좌/우 이동 화살표(<, >)를 추가하여 쉽게 넘겨볼 수 있습니다.
- **3D 여백 확보**: 전면 모드일 때 마지막 책의 표지가 3D 공간상으로 돌출되어 오른쪽 끝이 잘리는 현상을 막기 위해, 스크롤 컨테이너 우측에 넉넉한 140px의 투명 여백(Padding)을 확보했습니다.
- **Z-Index 계층 정리**: 좌우 화살표가 항상 팝업된 책들보다 최상단에 뜨도록 컴포넌트 간 겹침 순서를 확실하게 정리했습니다.

## 6. 파일 첨부 및 열람 기능
- **일반 파일 첨부 (Drag & Drop 및 파일 선택)**: 이미지 외에도 PDF, 문서, 엑셀 등 다양한 파일을 첨부할 수 있도록 데이터를 확장(`files`)했습니다.
- **파일 다운로드 UI**: 첨부된 일반 파일은 클립 아이콘과 함께 파일명, 용량(KB)이 표시된 카드 형태로 렌더링되며, 파일명을 클릭해 브라우저 단에서 즉시 다운로드 가능합니다.
- **모바일/PC 호환 파일 업로드 버튼**: 모바일에서는 OS 제약 상 드래그 앤 드롭이 불가능하므로, 책 패널 하단에 `<input type="file" multiple />` 기반의 명시적인 [파일 업로드] 버튼을 추가했습니다.

## 7. 책 패널 (BookPanel) UI/UX 개선
- **이미지 크게 보기 (Lightbox)**: 작은 썸네일 이미지를 클릭하면 어두운 배경과 함께 부드러운 애니메이션(`framer-motion`)으로 이미지가 화면 가득 확대되는 기능을 도입했습니다.
- **이미지 가로 스크롤 및 화살표**: 사진이 많아질 경우 지저분한 스크롤바를 숨기고, 컨테이너 양옆에 좌/우 둥근 화살표(Chevron)를 띄워 부드럽게 넘겨볼 수 있게 개선했습니다.
- **텍스트 박스 가시성 확보**: 책 내용을 적는 텍스트 입력창이 최소 10줄(250px) 이상 항상 확보되도록 레이아웃을 고정하여 긴 글도 편안하게 작성할 수 있습니다.
- **Z-Index 및 잘림 버그 해결**: 책 패널을 열었을 때 뒤쪽 책장의 화살표가 패널 위로 뚫고 나오는 버그를 고쳤으며, 첨부된 이미지의 'X(삭제)' 버튼이 컨테이너 여백 부족으로 잘리는 현상도 안쪽으로 배치하여 완벽히 수정했습니다.

## 8. 책 삭제 애니메이션 (블랙홀)
- **우측 하단 미니 휴지통 UI**: 책을 삭제할 때 기존의 거대한 사이드바 대신 우측 하단에 작고 직관적인 빨간색 원형(Trash) 블랙홀 공간을 마련했습니다.
- **빨려들어가는 이펙트**: 휴지통 구역 위에 책을 드롭하면 책이 0.5초 동안 작아지면서 0도로 누워 블랙홀 속으로 빨려 들어가는 유쾌한 물리 애니메이션을 적용했습니다.

## 9. 로컬 모바일 테스트 환경 구성
- **네트워크 호스팅**: Vite 개발 서버를 `--host` 옵션으로 실행하도록 구성하여, PC와 같은 Wi-Fi에 연결된 스마트폰에서도 접속해 모바일 접속을 즉시 지원하게 되었습니다.
